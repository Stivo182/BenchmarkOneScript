# Сортировка отчёта

По умолчанию строки в таблице отчёта следуют порядку объявления бенчмарков в классе. Для изменения порядка используйте одну из аннотаций или соответствующий метод API.

## Варианты сортировки

| Значение              | Описание                                                          |
|-----------------------|-------------------------------------------------------------------|
| `ПоУмолчанию`         | Порядок объявления бенчмарков в классе (используется по умолчанию) |
| `ОтБыстрыхКМедленным` | По возрастанию среднего времени выполнения (быстрые - первые)     |
| `ОтМедленныхКБыстрым` | По убыванию среднего времени выполнения (медленные - первые)      |
| `ПоМетоду`            | По алфавиту по имени метода бенчмарка                             |

## Конфигурирование

### Аннотации

```bsl
// Быстрые результаты - первые (сортировка по возрастанию времени)
&СортировкаОтБыстрыхКМедленным
Процедура ПриСозданииОбъекта()
	// ...
КонецПроцедуры
```

```bsl
// Медленные результаты - первые (сортировка по убыванию времени)
&СортировкаОтМедленныхКБыстрым
Процедура ПриСозданииОбъекта()
	// ...
КонецПроцедуры
```

```bsl
// Сортировка по имени метода бенчмарка (алфавитный порядок)
&СортировкаПоМетоду
Процедура ПриСозданииОбъекта()
	// ...
КонецПроцедуры
```

### API

Используйте метод `УстановитьСортировкуОтчета` объекта `КонфигурацияБенчмарков`:

```bsl
Конфигурация.УстановитьСортировкуОтчета(СортировкиОтчетаБенчмарков.ПоУмолчанию);         // Порядок объявления
Конфигурация.УстановитьСортировкуОтчета(СортировкиОтчетаБенчмарков.ОтБыстрыхКМедленным); // Быстрые - первые
Конфигурация.УстановитьСортировкуОтчета(СортировкиОтчетаБенчмарков.ОтМедленныхКБыстрым); // Медленные - первые
Конфигурация.УстановитьСортировкуОтчета(СортировкиОтчетаБенчмарков.ПоМетоду);            // По имени метода
```

## Пример

Два бенчмарка объявлены в порядке «Медленный, Быстрый». При включённой сортировке `ОтБыстрыхКМедленным` в отчёте они переупорядочиваются по возрастанию среднего времени.

```bsl
&Параметры(1, 2, 3)
Перем Множитель Экспорт;

&СортировкаОтБыстрыхКМедленным
Процедура ПриСозданииОбъекта()
КонецПроцедуры

&Бенчмарк
Процедура Медленный() Экспорт
	Приостановить(Множитель * 20);
КонецПроцедуры

&Бенчмарк
Процедура Быстрый() Экспорт
	Приостановить(Множитель * 10);
КонецПроцедуры
```
