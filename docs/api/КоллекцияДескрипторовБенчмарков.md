# КоллекцияДескрипторовБенчмарков

Коллекция дескрипторов бенчмарков. Может быть создана пустой или сразу заполнена бенчмарками указанного типа.

## Синтаксис:

```bsl
Новый КоллекцияДескрипторовБенчмарков(<Тип>)
```

**Параметры:**

| Имя | Тип | Значение по умолчанию | Описание |
| -- | -- | -- | -- |
| Тип | Тип, Неопределено | `Неопределено` | Тип класса, из которого нужно прочитать дескрипторы по аннотациям |


| Имя | Тип | Описание |
| -- | -- | -- |
| Дескриптор | [ДескрипторБенчмарка](ДескрипторБенчмарка.md) | Дескриптор, который нужно добавить |

## Фильтровать

**Синтаксис:**

```bsl
Фильтровать(<ФункцияФильтрации>, <ДополнительныеПараметры>)
```

**Параметры:**

| Имя | Тип | Значение по умолчанию | Описание |
| -- | -- | -- | -- |
| ФункцияФильтрации | Строка |  | Лямбда-выражение фильтрации. Получает элемент коллекции и возвращает `Истина`, если элемент нужно оставить. |
| ДополнительныеПараметры | Структура | `Неопределено` | Контекст добавляемый в лямбда выражение |

**Описание:**

Оставляет элементы, для которых функция фильтрации вернула `Истина`

## Пример

```bsl

Дескриптор = Дескрипторы.НайтиПоИмени("ПоискВМассиве");

Если Дескриптор <> Неопределено Тогда
    Результат = Бенчмаркинг.Запустить(Дескриптор);
КонецЕсли;
```
