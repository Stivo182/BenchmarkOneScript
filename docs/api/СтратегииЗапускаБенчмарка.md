# СтратегииЗапускаБенчмарка

## Синтаксис:

```bsl
СтратегииЗапускаБенчмарка.<Значение>
```

## Значения

[ПропускнаяСпособность](#пропускнаяспособность) </br>
[ХолодныйЗапуск](#холодныйзапуск)


## ПропускнаяСпособность

**Синтаксис:**

```bsl
СтратегииЗапускаБенчмарка.ПропускнаяСпособность
```

**Тип значения:**

Строка.

**Описание:**

Автоматически рассчитывает число вызовов за итерацию и выполняет прогрев.


## ХолодныйЗапуск

**Синтаксис:**

```bsl
СтратегииЗапускаБенчмарка.ХолодныйЗапуск
```

**Тип значения:**

Строка.

**Описание:**

Запускает один вызов за итерацию без прогрева.
