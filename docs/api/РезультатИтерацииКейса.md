# РезультатИтерацииКейса

Замер одной итерации кейса.

## Синтаксис:

```bsl
Новый РезультатИтерацииКейса()
```

## Поля

| Имя | Тип | Значение по умолчанию | Описание |
| -- | -- | -- | -- |
| Этап | Строка | `""` | Этап запуска бенчмарка. см. [ЭтапыБенчмарка](ЭтапыБенчмарка.md) |
| НомерИтерации | Число | `0` | Номер итерации |
| КоличествоОпераций | Число | `0` | Количество вызовов метода в итерации |
| Наносекунд | Число | `0` | Длительность итерации |
| НаносекундЗаОперацию | Число | `0` | Длительность одного вызова |
| ВыделяемаяПамять | Число | `0` | Байт выделяемой памяти за итерацию |
| ВыделяемаяПамятьЗаОперацию | Число | `0` | Байт выделяемой памяти за один вызов |
