# ПараметрБенчмарка

Параметр бенчмарка: имя, значение, строковое представление и признак параметра метода.

## Синтаксис:

```bsl
Новый ПараметрБенчмарка(<Имя>, <Значение>, <ЭтоПараметрМетода>)
```

**Параметры:**

| Имя | Тип | Значение по умолчанию | Описание |
| -- | -- | -- | -- |
| Имя | Строка | `""` | Имя параметра |
| Значение | Произвольный | `Неопределено` | Значение параметра |
| ЭтоПараметрМетода | Булево | `Ложь` | `Истина`, если параметр передается в метод бенчмарка |
