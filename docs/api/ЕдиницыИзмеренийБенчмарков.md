## Создать

**Синтаксис:**

```bsl
Создать(<Обозначение>, <Наименование>, <Базовая>, <ОтношениеКБазовой>, <ВыводитьВЛегенде>)
```

**Параметры:**

| Имя | Тип | Значение по умолчанию | Описание |
| -- | -- | -- | -- |
| Обозначение | Строка |  | Краткое обозначение, например `ms` или `KB` |
| Наименование | Строка | `""` | Полное наименование |
| Базовая | [ЕдиницаИзмеренияБенчмарка](ЕдиницаИзмеренияБенчмарка.md), Неопределено | `Неопределено` | Базовая единица той же размерности |
| ОтношениеКБазовой | Число | `1` | Количество базовых единиц в данной единице |
| ВыводитьВЛегенде | Булево | `Истина` | Включать единицу в легенду отчета |

**Возвращаемое значение:**

Тип: [ЕдиницаИзмеренияБенчмарка](ЕдиницаИзмеренияБенчмарка.md).

**Описание:**

Создает и регистрирует новую единицу.


**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |

## Пример

```bsl

    1500000,
    ЕдиницыИзмеренийБенчмарков.Наносекунда,
    ЕдиницыИзмеренийБенчмарков.Миллисекунда
);
```
