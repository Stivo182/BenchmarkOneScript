# СортировкиОтчетаБенчмарков

## Синтаксис:

```bsl
СортировкиОтчетаБенчмарков.<Значение>
```

## Значения

[ПоУмолчанию](#поумолчанию) </br>
[ОтБыстрыхКМедленным](#отбыстрыхкмедленным) </br>
[ОтМедленныхКБыстрым](#отмедленныхкбыстрым) </br>
[Метод](#метод)


## ПоУмолчанию

**Синтаксис:**

```bsl
СортировкиОтчетаБенчмарков.ПоУмолчанию
```

**Тип значения:**

Строка.

**Описание:**

Сохраняет порядок построения отчета.


## ОтБыстрыхКМедленным

**Синтаксис:**

```bsl
СортировкиОтчетаБенчмарков.ОтБыстрыхКМедленным
```

**Тип значения:**

Строка.

**Описание:**

Сортирует от быстрых кейсов к медленным.


## ОтМедленныхКБыстрым

**Синтаксис:**

```bsl
СортировкиОтчетаБенчмарков.ОтМедленныхКБыстрым
```

**Тип значения:**

Строка.

**Описание:**

Сортирует от медленных кейсов к быстрым.


## Метод
