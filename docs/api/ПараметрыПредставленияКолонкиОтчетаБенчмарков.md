# ПараметрыПредставленияКолонкиОтчетаБенчмарков

Параметры отображения отдельной колонки отчета.

## Синтаксис:

```bsl
Новый ПараметрыПредставленияКолонкиОтчетаБенчмарков()
```

## Поля

| Имя | Тип | Значение по умолчанию | Описание |
| -- | -- | -- | -- |
| ЕдиницаИзмеренияПредставления | [ЕдиницаИзмеренияБенчмарка](ЕдиницаИзмеренияБенчмарка.md), Неопределено | `Неопределено` | Единица, в которой выводится значение |
| ФорматнаяСтрока | Строка | `""` | Формат числа |
| Скрыта | Булево | `Ложь` | Признак скрытия колонки |
| Размер | Число | `0` | Ширина колонки для консольного вывода |
