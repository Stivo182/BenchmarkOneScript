# ИсполняющаяСредаБенчмарков

Описание версии OneScript, в которой нужно выполнить бенчмарки.

## Синтаксис:

```bsl
Новый ИсполняющаяСредаБенчмарков()
```

## Поля

| Имя | Тип | Значение по умолчанию | Описание |
| -- | -- | -- | -- |
| Версия | Строка | `""` | Версия OneScript |
| Наименование | Строка | `""` | Имя для отображения в отчете |
| ЭтоЭталон | Булево | `Ложь` | Признак эталонной среды |

## Использование

Обычно экземпляры создаются через [КонфигурацияБенчмарков](КонфигурацияБенчмарков.md):

```bsl
Конфигурация = Новый КонфигурацияБенчмарков()
    .ДобавитьИсполняющуюСреду("1.9.3", "stable", Истина)
    .ДобавитьИсполняющуюСреду("2.0.0", "next");
```
