# ЧекпоинтЗапускаБенчмарков

Чекпоинт изолированного запуска. Используется библиотекой для продолжения выполнения с определенного кейса, этапа и итерации.

## Синтаксис:

```bsl
Новый ЧекпоинтЗапускаБенчмарков()
```

## Поля

| Имя | Тип | Значение по умолчанию | Описание |
| -- | -- | -- | -- |
| ИмяМетода | Строка | `""` | Имя метода бенчмарка |
| ИдентификаторКейса | Число | `0` | Идентификатор кейса |
| Этап | Строка | `""` | Этап запуска бенчмарка. см. [ЭтапыБенчмарка](ЭтапыБенчмарка.md) |
| НомерИтерации | Число | `0` | Номер итерации |
