# СериализацияОбъектовБенчмаркинга

Публичные API для записи и чтения JSON-представления настроек и результатов бенчмарков.

## Методы

[ВJson](#вjson) </br>
[ИзJson](#изjson) </br>
[ПривестиКПримитиву](#привестикпримитиву) </br>
[ЭтоПримитив](#этопримитив)


## ВJson

**Синтаксис:**

```bsl
ВJson(<Значение>)
```

**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |
| Значение | Произвольный | Значение, которое требуется сериализовать в JSON. |

**Возвращаемое значение:**

Тип: Строка.

**Описание:**

Сериализует значение в JSON.

## ИзJson

**Синтаксис:**

```bsl
ИзJson(<Json>, <ТипОбъекта>)
```

**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |
| Json | Строка | Строка JSON. |
| ТипОбъекта | Тип | Тип объекта, в который необходимо выполнить десериализацию. |

**Возвращаемое значение:**

Тип: Произвольный.

**Описание:**

Десериализует JSON в объект указанного типа.

## ПривестиКПримитиву

**Синтаксис:**

```bsl
ПривестиКПримитиву(<Значение>)
```

**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |
| Значение | Произвольный | Любое значение для обработки |

**Возвращаемое значение:**

Тип: Произвольный.

**Описание:**

Возвращает примитивное значение как есть, составное значение приводит к строке.

## ЭтоПримитив

**Синтаксис:**

```bsl
ЭтоПримитив(<Значение>)
```

**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |
| Значение | Произвольный | Проверяемое значение |

**Возвращаемое значение:**

Тип: Булево.

**Описание:**

Проверяет, относится ли значение к примитивным типам.
