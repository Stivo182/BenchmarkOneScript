# НастройкиЗапускаБенчмарков

Сериализуемый контейнер настроек изолированного запуска. Обычно создается библиотекой для передачи данных в дочерний процесс.

## Синтаксис:

```bsl
Новый НастройкиЗапускаБенчмарков()
```

## Поля

| Имя | Тип | Значение по умолчанию | Описание |
| -- | -- | -- | -- |
| ДескрипторыБенчмарков | [КоллекцияДескрипторовБенчмарков](КоллекцияДескрипторовБенчмарков.md) | | Дескрипторы для запуска |
| Конфигурация | [КонфигурацияБенчмарков](КонфигурацияБенчмарков.md) | | Конфигурация запуска |
| ИдентификаторыКейсов | Массив из Число, Неопределено | `Неопределено` | Идентификаторы кейсов, которые нужно выполнить |
| Чекпоинт | [ЧекпоинтЗапускаБенчмарков](ЧекпоинтЗапускаБенчмарков.md), Неопределено | `Неопределено` | Чекпоинт продолжения запуска |
