# ЭкспортерРезультатовБенчмарков

Класс-координатор экспорта результатов. Обходит экспортеры из конфигурации и записывает файлы в каталог артефактов.

## Синтаксис:

```bsl
Новый ЭкспортерРезультатовБенчмарков(<Репортер>)
```

**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |
| Репортер | Произвольный | Объект репортера, который выводит сообщения об экспорте |

## Методы

[Записать](#записать)


## Записать

**Синтаксис:**

```bsl
Записать(<РезультатЗапуска>, <Каталог>)
```

**Параметры:**

| Имя | Тип | Значение по умолчанию | Описание |
| -- | -- | -- | -- |
| РезультатЗапуска | [РезультатЗапускаБенчмарков](РезультатЗапускаБенчмарков.md) |  | Результат, для которого нужно записать артефакты |
| Каталог | Строка | `""` | Каталог артефактов. Если не передан, используется каталог из конфигурации |

**Описание:**

Записывает артефакты для [РезультатЗапускаБенчмарков](РезультатЗапускаБенчмарков.md).
