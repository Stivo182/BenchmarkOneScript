# СериализаторНастроекЗапускаБенчмарков

Класс для записи и чтения настроек запуска в JSON.

## Синтаксис:

```bsl
Новый СериализаторНастроекЗапускаБенчмарков()
```

## Методы

[ЗаписатьВJson](#записатьвjson) </br>
[ПрочитатьИзJson](#прочитатьизjson)


## ЗаписатьВJson

**Синтаксис:**

```bsl
ЗаписатьВJson(<НастройкиЗапуска>, <ИмяФайла>)
```

**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |
| НастройкиЗапуска | [НастройкиЗапускаБенчмарков](НастройкиЗапускаБенчмарков.md) | Настройки запуска |
| ИмяФайла | Строка | Имя файла, в который будет записаны настройки |

**Описание:**

Записывает [НастройкиЗапускаБенчмарков](НастройкиЗапускаБенчмарков.md) в JSON-файл.

## ПрочитатьИзJson

**Синтаксис:**

```bsl
ПрочитатьИзJson(<ИмяФайла>)
```

**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |
| ИмяФайла | Строка | Имя файла, из которого будет прочитаны настройки. |

**Возвращаемое значение:**

Тип: [НастройкиЗапускаБенчмарков](НастройкиЗапускаБенчмарков.md).

**Описание:**

Читает настройки запуска из JSON-файла.
