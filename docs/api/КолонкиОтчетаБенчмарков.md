# КолонкиОтчетаБенчмарков

Модуль `КолонкиОтчетаБенчмарков` содержит имена стандартных колонок и фабрики объектов `КолонкаОтчетаБенчмарков`.

## Имена колонок

| Имя | Описание |
| -- | -- |
| `Метод` | Имя метода бенчмарка |
| `Среднее` | Среднее время выполнения |
| `СтандартноеОтклонение` | Стандартное отклонение |
| `СтандартнаяОшибка` | Стандартная ошибка среднего |
| `Медиана` | Медиана |
| `НижнийКвартиль` | Нижний квартиль |
| `ВерхнийКвартиль` | Верхний квартиль |
| `ОперацийВСекунду` | Производительность |
| `Эталон` | Признак эталонного кейса |
| `КоэффициентПроизводительности` | Коэффициент относительно эталона |
| `RatioSD` | Отношение стандартного отклонения к среднему |
| `Категория` | Категория бенчмарка |
| `ВыделяемаяПамять` | Выделяемая память |
| `Мин` | Минимальное время |
| `Макс` | Максимальное время |
| `Процентиль` | Префикс колонок процентилей |
| `ВерсияИсполняющейСреды` | Версия runtime |
