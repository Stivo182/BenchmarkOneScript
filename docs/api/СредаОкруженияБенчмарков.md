# СредаОкруженияБенчмарков

Сведения об окружении, в котором выполнялся запуск.

## Синтаксис:

```bsl
Новый СредаОкруженияБенчмарков()
```

## Поля

| Имя | Тип |  Описание |
| -- | -- | -- |
| ВерсияБиблиотеки | Строка |Версия пакета `benchmark` |
| ВерсияОС | Строка | Версия операционной системы |
| ВерсияИсполняющейСреды | Строка |  Версия OneScript |
| ИнформацияОПроцессоре | ИнформацияОПроцессоре | Данные процессора из `cpuinfo` |

> [!NOTE]
> Имя процессора, количество процессоров, ядер и логических процессоров сериализуются в JSON результата, но в объектной модели доступны через поле `ИнформацияОПроцессоре`.
