# РезультатЗапускаКейса

Результат выполнения одного кейса бенчмарка.

## Синтаксис:

```bsl
Новый РезультатЗапускаКейса()
```

## Поля

| Имя | Тип | Значение по умолчанию | Описание |
| -- | -- | -- | -- |
| Кейс | [КейсБенчмарка](КейсБенчмарка.md), Неопределено | `Неопределено` | Выполненный кейс |
| Эталон | [РезультатЗапускаКейса](РезультатЗапускаКейса.md), Неопределено | `Неопределено` | Эталонный результат для сравнения |
| ЭтоЭталон | Булево | `Ложь` | Признак эталонного кейса |
| Статистика | [СтатистикаБенчмарка](СтатистикаБенчмарка.md) | | Рассчитанная статистика |
| Замеры | Массив из [РезультатИтерацииКейса](РезультатИтерацииКейса.md) | | Замеры отдельных итераций |

## Методы

[НайтиИтерациюЗамера](#найтиитерациюзамера)


## НайтиИтерациюЗамера

**Синтаксис:**

```bsl
НайтиИтерациюЗамера(<Этап>, <НомерИтерации>)
```

**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |
| Этап | Строка | Этап запуска бенчмарка. см. [ЭтапыБенчмарка](ЭтапыБенчмарка.md) |
| НомерИтерации | Число | Номер итерации |

**Возвращаемое значение:**

Тип: [РезультатИтерацииКейса](РезультатИтерацииКейса.md), Неопределено.

**Описание:**

Ищет замер по этапу и номеру итерации.
