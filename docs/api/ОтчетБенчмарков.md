## Добавить

**Синтаксис:**

```bsl

```

**Параметры:**

| Имя | Тип | Значение по умолчанию | Описание |
| -- | -- | -- | -- |
| РезультатКейса | [РезультатЗапускаКейса](РезультатЗапускаКейса.md), Неопределено | `Неопределено` | Результат запуска кейса, связанный с добавляемой строкой отчета. |

**Возвращаемое значение:**

Тип: СтрокаТаблицыЗначений.

**Описание:**

Добавляет строку в табличные данные отчета.

## РезультатЗапускаКейсаСтроки

| Имя | Тип | Описание |
