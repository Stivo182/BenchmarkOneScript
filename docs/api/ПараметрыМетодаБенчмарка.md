# ПараметрыМетодаБенчмарка

Коллекция параметров, которые передаются в метод бенчмарка в порядке добавления.

## Синтаксис:

```bsl
Новый ПараметрыМетодаБенчмарка()
```

## Методы

[Добавить](#добавить) </br>
[Количество](#количество) </br>
[Получить](#получить) </br>
[ВМассив](#вмассив)


## Добавить

**Синтаксис:**

```bsl
Добавить(<Параметр>)
```

**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |
| Параметр | [ПараметрБенчмарка](ПараметрБенчмарка.md) | Параметр метода, который нужно добавить. |

**Описание:**

Добавляет [ПараметрБенчмарка](ПараметрБенчмарка.md).

## Количество

**Синтаксис:**

```bsl
Количество()
```

**Возвращаемое значение:**

Тип: Число.

**Описание:**

Возвращает количество параметров.

## Получить

**Синтаксис:**

```bsl
Получить(<Индекс>)
```

**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |
| Индекс | Число | Индекс параметра, начиная с 0. |

**Возвращаемое значение:**

Тип: [ПараметрБенчмарка](ПараметрБенчмарка.md).

**Описание:**

Возвращает параметр по индексу.

## ВМассив

**Синтаксис:**

```bsl
ВМассив()
```

**Возвращаемое значение:**

Тип: Массив из [ПараметрБенчмарка](ПараметрБенчмарка.md).

**Описание:**

Возвращает параметры массивом.
