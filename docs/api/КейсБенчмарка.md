# КейсБенчмарка

Кейс запуска: дескриптор бенчмарка, набор параметров и исполняющая среда.

## Синтаксис:

```bsl
Новый КейсБенчмарка(<Идентификатор>, <Дескриптор>, <Параметры>, <ИсполняющаяСреда>)
```

**Параметры:**

| Имя | Тип | Значение по умолчанию | Описание |
| -- | -- | -- | -- |
| Идентификатор | Число |  | Уникальный идентификатор кейса в запуске |
| Дескриптор | [ДескрипторБенчмарка](ДескрипторБенчмарка.md) |  | Дескриптор метода бенчмарка |
| Параметры | Массив из [ПараметрБенчмарка](ПараметрБенчмарка.md), Неопределено | `Неопределено` | Параметры запуска кейса. Если не заданы, создается пустой массив параметров |
| ИсполняющаяСреда | [ИсполняющаяСредаБенчмарков](ИсполняющаяСредаБенчмарков.md), Неопределено | `Неопределено` | Runtime для запуска |
