# СериализаторРезультатовБенчмарков

Класс для записи и чтения результата запуска в JSON.

## Синтаксис:

```bsl
Новый СериализаторРезультатовБенчмарков()
```

## Методы

[ЗаписатьВJson](#записатьвjson) </br>
[ПрочитатьИзJson](#прочитатьизjson)


## ЗаписатьВJson

**Синтаксис:**

```bsl
ЗаписатьВJson(<РезультатЗапуска>, <ПолноеИмяФайла>)
```

**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |
| РезультатЗапуска | [РезультатЗапускаБенчмарков](РезультатЗапускаБенчмарков.md) | Результат запуска, который нужно записать в JSON. |
| ПолноеИмяФайла | Строка | Полный путь к файлу, в который будут записаны результат запуска бенчмарков |

**Описание:**

Записывает [РезультатЗапускаБенчмарков](РезультатЗапускаБенчмарков.md) в JSON-файл.

## ПрочитатьИзJson

**Синтаксис:**

```bsl
ПрочитатьИзJson(<ПолноеИмяФайла>)
```

**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |
| ПолноеИмяФайла | Строка | Полный путь к JSON-файлу с результатами бенчмарков |

**Возвращаемое значение:**

Тип: [РезультатЗапускаБенчмарков](РезультатЗапускаБенчмарков.md).

**Описание:**

Читает результат запуска из JSON-файла.
