# СтатистикаБенчмарка

Класс с агрегированной статистикой по замерам кейса.

## Синтаксис:

```bsl
Новый СтатистикаБенчмарка(<Замеры>)
```

**Параметры:**

| Имя | Тип | Значение по умолчанию | Описание |
| -- | -- | -- | -- |
| Замеры | Массив из [РезультатИтерацииКейса](РезультатИтерацииКейса.md), Структура, Неопределено | `Неопределено` | Замеры, по которым нужно сразу рассчитать статистику. Структура должна содержать поля `ЗамерыВремени` и `ЗамерыПамяти` |

## Поля

| Имя | Тип | Описание |
| -- | -- | -- |
| КоличествоИтераций | Число | Количество измерительных итераций |
| Мин | Число | Минимальное время |
| НижнийКвартиль | Число | Нижний квартиль |
| Медиана | Число | Медиана |
| Среднее | Число | Среднее значение |
| ВерхнийКвартиль | Число | Верхний квартиль |
| Макс | Число | Максимальное время |
| СтандартноеОтклонение | Число | Стандартное отклонение |
| СтандартнаяОшибкаСреднего | Число | Стандартная ошибка среднего |
| ОперацийВСекунду | Число | Производительность |
| ВыделяемаяПамять | Число | Выделяемая память |

## Квантиль

**Синтаксис:**

```bsl
Квантиль(<Вероятность>)
```

**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |
| Вероятность | Число | Вероятность квантиля от 0 до 1. |

**Возвращаемое значение:**

Тип: Число.

**Описание:**

Возвращает квантиль времени выполнения.

## Прочитать

**Синтаксис:**

```bsl

```

**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |
| Замеры | Массив из [РезультатИтерацииКейса](РезультатИтерацииКейса.md), Структура | Замеры, по которым нужно пересчитать статистику. Можно передать массив итераций или структуру с замерами времени и памяти. |

**Возвращаемое значение:**

Тип: [СтатистикаБенчмарка](СтатистикаБенчмарка.md).

**Описание:**

Пересчитывает статистику по замерам.
