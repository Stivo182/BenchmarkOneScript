# КолонкаОтчетаБенчмарков

Описание колонки отчета: имя, заголовок, единица измерения, делегаты вычисления и параметры представления.

## Синтаксис:

```bsl
Новый КолонкаОтчетаБенчмарков(<ИмяКолонки>)
```

**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |
| ИмяКолонки | Строка | Внутреннее имя колонки |

## Поля

| Имя | Тип | Значение по умолчанию | Описание |
| -- | -- | -- | -- |
| Имя | Строка |  | Внутреннее имя колонки |
| Заголовок | Строка | `""` | Заголовок в отчете |
| Описание | Строка | `""` | Описание для легенды |
| ЕдиницаИзмерения | [ЕдиницаИзмеренияБенчмарка](ЕдиницаИзмеренияБенчмарка.md), Неопределено | `Неопределено` | Базовая единица измерения |
| СтильКоэффициента | `Строка` | `СтилиКоэффициентаБенчмарка.Значение` | Стиль представления коэффициентов |
| ПараметрыПредставления | [ПараметрыПредставленияКолонкиОтчетаБенчмарков](ПараметрыПредставленияКолонкиОтчетаБенчмарков.md) |  | Настройки отображения |
| ДелегатЗначение | Действие, Неопределено | `Неопределено` | Делегат вычисления значения |
| ДелегатПредставлениеЗначения | Действие, Неопределено | `Неопределено` | Делегат пользовательского представления |
| ФункцияСкрытия | Строка | `""` | Лямбда-условие автоматического скрытия |

## Значение

**Синтаксис:**

```bsl
Значение(<РезультатКейса>, <Колонка>)
```

**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |
| РезультатКейса | [РезультатЗапускаКейса](РезультатЗапускаКейса.md) | Результат кейса, для которого вычисляется значение. |
| Колонка | [КолонкаОтчетаБенчмарков](КолонкаОтчетаБенчмарков.md) | Описание колонки, передаваемое делегату вычисления. |

**Возвращаемое значение:**

Тип: Произвольный.

**Описание:**

Вычисляет значение колонки для [РезультатЗапускаКейса](РезультатЗапускаКейса.md).

## ПредставлениеЗначения

**Синтаксис:**

```bsl
ПредставлениеЗначения(<Значение>, <РезультатКейса>, <ПараметрыПредставления>)
```

**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |
| Значение | Произвольный | Значение колонки |
| РезультатКейса | [РезультатЗапускаКейса](РезультатЗапускаКейса.md), Неопределено | Результат кейса для контекста представления. |
| ПараметрыПредставления | [ПараметрыПредставленияКолонкиОтчетаБенчмарков](ПараметрыПредставленияКолонкиОтчетаБенчмарков.md), Неопределено | Параметры форматирования и отображения значения. |

**Возвращаемое значение:**

Тип: Строка.

**Описание:**

Возвращает строковое представление значения с учетом [ПараметрыПредставленияКолонкиОтчетаБенчмарков](ПараметрыПредставленияКолонкиОтчетаБенчмарков.md).
