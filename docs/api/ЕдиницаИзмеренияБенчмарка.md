# ЕдиницаИзмеренияБенчмарка

Единица измерения для значений отчета: время, память или производительность.

## Синтаксис:

```bsl
Новый ЕдиницаИзмеренияБенчмарка(
    <Обозначение>,
    <Наименование>,
    <Базовая>,
    <ОтношениеКБазовой>,
    <ВыводитьВЛегенде>)
```

**Параметры:**

| Имя | Тип | Значение по умолчанию | Описание |
| -- | -- | -- | -- |
| Обозначение | Строка |  | Краткое обозначение |
| Наименование | Строка | `""` | Полное имя |
| Базовая | [ЕдиницаИзмеренияБенчмарка](ЕдиницаИзмеренияБенчмарка.md), Неопределено | `Неопределено` | Базовая единица размерности |
| ОтношениеКБазовой | Число | `1` | Коэффициент пересчета в базовую единицу |
| ВыводитьВЛегенде | Булево | `Истина` | Добавлять единицу в легенду отчета |
