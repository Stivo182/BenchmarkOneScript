# ОбработчикСобытияБенчмарка

Описание связи события жизненного цикла бенчмарка и метода-обработчика.

## Синтаксис:

```bsl
Новый ОбработчикСобытияБенчмарка(<КвалифицированноеИмяМетода>, <Событие>)
```

**Параметры:**

| Имя | Тип | Значение по умолчанию | Описание |
| -- | -- | -- | -- |
| КвалифицированноеИмяМетода | Строка | `""` | Имя метода в формате `<ИмяМетода>`, `<ИмяМодуля>.<ИмяМетода>` или `<ИмяТипа>.<ИмяМетода>` |
| Событие | Строка | `""` | Событие жизненного цикла бенчмарков. см. [СобытияБенчмарков](СобытияБенчмарков.md) |

## Событие

**Синтаксис:**

```bsl
Событие()
```

**Возвращаемое значение:**

Тип: Строка.

**Описание:**

Возвращает событие обработчика. см. [СобытияБенчмарков](СобытияБенчмарков.md)

## ИмяМетода

**Синтаксис:**

```bsl
ИмяМетода()
```

**Возвращаемое значение:**

Тип: Строка.

**Описание:**

Возвращает квалифицированное имя метода.

## Делегат

**Синтаксис:**

```bsl
Делегат(<Объект>)
```

**Параметры:**

| Имя | Тип | Значение по умолчанию | Описание |
| -- | -- | -- | -- |
| Объект | Произвольный | `Неопределено` | Объект для создания делегата. |

**Возвращаемое значение:**

Тип: Действие.

**Описание:**

Создает и кэширует делегат вызова метода-обработчика.
