# РезультатЗапускаБенчмарков

Результат полного запуска бенчмарков. Возвращается методом [Запустить](Бенчмаркинг.md#запустить).

## Синтаксис:

```bsl
Новый РезультатЗапускаБенчмарков()
```

## Поля

| Имя | Тип | Описание |
| -- | -- | -- |
| СредаОкружения | [СредаОкруженияБенчмарков](СредаОкруженияБенчмарков.md) | Сведения об окружении запуска |
| Конфигурация | [КонфигурацияБенчмарков](КонфигурацияБенчмарков.md) | Конфигурация, с которой выполнялся запуск |
| ДескрипторыБенчмарков | [КоллекцияДескрипторовБенчмарков](КоллекцияДескрипторовБенчмарков.md) | Запущенные дескрипторы |
| Запуски | Массив из [РезультатЗапускаКейса](РезультатЗапускаКейса.md) | Результаты кейсов |
| Чекпоинт | [ЧекпоинтЗапускаБенчмарков](ЧекпоинтЗапускаБенчмарков.md), Неопределено | Чекпоинт изолированного запуска |
| Отчет | [ОтчетБенчмарков](ОтчетБенчмарков.md)  | Сформированный отчет |

## УстановитьСвязиСЭталонами

**Синтаксис:**

```bsl
УстановитьСвязиСЭталонами()
```

**Описание:**

Связывает результаты кейсов с эталонами.


## ПересчитатьСтатистику

**Синтаксис:**

```bsl
ПересчитатьСтатистику()
```

**Описание:**

Пересчитывает статистику всех кейсов.


## СформироватьОтчет

**Синтаксис:**

```bsl
СформироватьОтчет()
```

**Описание:**

Строит отчет по результатам запуска.

## Пример

```bsl
Результат = Бенчмаркинг.Запустить(Тип("БенчмаркСтрок"));

Сообщить(Результат.СредаОкружения.ВерсияИсполняющейСреды);
Сообщить(Результат.Отчет.КакMarkdown());
```
