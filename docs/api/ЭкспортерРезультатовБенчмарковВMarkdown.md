# ЭкспортерРезультатовБенчмарковВMarkdown

Экспортер отчета в Markdown.

## Синтаксис:

```bsl
Новый ЭкспортерРезультатовБенчмарковВMarkdown()
```

## Методы

[Записать](#записать) </br>
[Расширение](#расширение)


## Записать

**Синтаксис:**

```bsl
Записать(<РезультатЗапуска>, <ПолноеИмяФайла>)
```

**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |
| РезультатЗапуска | [РезультатЗапускаБенчмарков](РезультатЗапускаБенчмарков.md) | Результат запуска для экспорта |
| ПолноеИмяФайла | Строка | Полное имя файла |

**Описание:**

Записывает Markdown-отчет по [РезультатЗапускаБенчмарков](РезультатЗапускаБенчмарков.md).

## Расширение

**Синтаксис:**

```bsl
Расширение()
```

**Возвращаемое значение:**

Тип: Строка.

**Описание:**

Возвращает расширение файла.
