## Синтаксис:

```bsl
Новый ДескрипторБенчмарка(<Тип>, <Метод>)
```

**Параметры:**

| Имя | Тип | Значение по умолчанию | Описание |
| -- | -- | -- | -- |
| Тип | Тип, Неопределено | `Неопределено` | Тип класса бенчмарков |
| Метод | Строка | `""` | Имя метода бенчмарка |

## Пример

```bsl
ПараметрыМетода = Новый Массив();
ПараметрыМетода.Добавить(1000);

Дескриптор = Новый ДескрипторБенчмарка(Тип("БенчмаркСтрок"), "Конкатенация")
    .УстановитьКатегорию("Строки")
    .УстановитьНаименование("Конкатенация через +")
    .ДобавитьПараметры(ПараметрыМетода);

Результат = Бенчмаркинг.Запустить(Дескриптор);
```
