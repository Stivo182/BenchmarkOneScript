# Примеры бенчмарков

Каталог содержит `.os`-файлы с классами бенчмарков. Его можно подключить как библиотеку:

```bsl
#Использовать "./samples/benchmarks"
```

## Разделы

- [examples](examples/) - прикладные и исследовательские бенчмарки для сравнения подходов и механизмов OneScript.
- [features](features/) - короткие примеры возможностей библиотеки: параметры, категории, эталоны, обработчики, колонки отчета и экспорт.

## Запуск

Запуск одного файла:

```bash
benchos run samples/benchmarks/examples/БенчмаркСборкиСтрок.os
```

Запуск всех бенчмарков раздела:

```bash
benchos run samples/benchmarks/features
```

Запуск из кода:

```bsl
#Использовать benchmark
#Использовать "./samples/benchmarks"

Бенчмаркинг.Запустить(Тип("БенчмаркБазовый"));
```
