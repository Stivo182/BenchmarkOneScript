## Практические и исследовательские бенчмарки

В данном каталоге размещены прикладные и исследовательские бенчмарки, позволяющие получить полезные измеримые результаты о работе механизмов движка и различных подходах к реализации.

### Запуск бенчмарков

Для запуска используйте команду:

```bash
benchos run [OPTIONS] [PATH]
```

Подробное описание параметров и доступных опций приведено в [документации CLI](../../../docs/CLI.md).

### Примеры запуска

```bash
# Запуск конкретного файла с бенчмарками
benchos run ./БенчмаркБазовый.os

# Запуск всех бенчмарков в текущем каталоге
benchos run .
```
