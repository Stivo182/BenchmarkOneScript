# Обработчики событий

Каталог содержит пример класса с обработчиками событий жизненного цикла запуска бенчмарков.

## Состав

- [ОбработчикиСобытийБенчмарковПример.os](Классы/ОбработчикиСобытийБенчмарковПример.os) - экспортирует процедуры `Инициализация`, `Завершение`, `ПередЗапуском` и `ПослеЗапуска`.

## Использование

Класс можно подключить через `#Использовать` и передать методы обработчиков в конфигурацию запуска:

```bsl
#Использовать "../handlers"

Конфигурация.ДобавитьОбработчикСобытия(
	"ОбработчикиСобытийБенчмарковПример.Инициализация",
	СобытияБенчмарков.ПередВсеми);
```
