# Пользовательские колонки отчета

Каталог содержит модули пользовательских колонок для итогового отчета бенчмарков.

## Состав

- [КолонкаОтчетаБенчмарковСреднееВМиллисекундах.os](Модули/КолонкаОтчетаБенчмарковСреднееВМиллисекундах.os) - выводит среднее время выполнения в миллисекундах.
- [КолонкаОтчетаБенчмарковРазмахВремениВМиллисекундах.os](Модули/КолонкаОтчетаБенчмарковРазмахВремениВМиллисекундах.os) - выводит размах времени между максимумом и минимумом.

Каждый модуль экспортирует функции `Создать()` и `Значение(...)`, которые используются библиотекой при построении отчета.

## Использование

Подключите каталог и укажите имя колонки в аннотации `&Колонка`:

```bsl
#Использовать "../../customColumns"

&Колонка("СреднееВМиллисекундах")
Процедура ПриСозданииОбъекта()
КонецПроцедуры
```
