# Примеры

В этом каталоге собраны готовые сценарии использования `BenchmarkOneScript`: запуск бенчмарков через API, самостоятельные файлы бенчмарков и подключаемые расширения для параметров, обработчиков событий и колонок отчета.

## Разделы

- [api](api/) - примеры программного запуска и настройки бенчмарков через публичный API.
- [benchmarks](benchmarks/) - готовые классы бенчмарков для запуска через CLI или API.
- [customColumns](customColumns/) - модули пользовательских колонок отчета.
- [handlers](handlers/) - класс обработчиков событий жизненного цикла запуска.
- [paramsSources](paramsSources/) - классы внешних источников параметров.
