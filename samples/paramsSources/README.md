# Источники параметров

Каталог содержит примеры классов, которые возвращают наборы параметров для бенчмарков.

## Состав

- [КоллекцияМножителей.os](Классы/КоллекцияМножителей.os) - возвращает массив множителей.
- [ПараметрыБенчмаркаФакториала.os](Классы/ПараметрыБенчмаркаФакториала.os) - возвращает таблицу значений с параметрами для сравнения алгоритмов факториала.

## Использование

Подключите каталог и укажите источник параметров в аннотации `&ИсточникПараметров`:

```bsl
#Использовать "../../paramsSources"

&ИсточникПараметров("ПараметрыБенчмаркаФакториала.Получить")
Процедура БенчмаркФакториала(Число, ТипАлгоритма) Экспорт
КонецПроцедуры
```
